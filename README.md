# 🕵️ AI-Haklab Pro v17.5: Tactical Hacker Engine
> **El primer agente de IA nativo para Termux, diseñado para la vanguardia del Pentesting Móvil.**

## 🌐 Contexto del Proyecto
AI-Haklab Pro es un motor de inteligencia táctica desarrollado por **@kuromi04** (The Creator). Este proyecto nace como una herramienta avanzada para los estudiantes de la comunidad **ivan3bycinderella**, utilizando el ecosistema de herramientas y laboratorios de **i-Haklab** creado por **ivam3**.

---

Este repositorio representa el **primer agente de Pentesting con IA que corre de forma nativa en Termux**. 

*   **Creador & Lead Dev:** [@kuromi04](https://github.com/kuromi04)

---

## 🚀 Filosofía Didáctica (Hacker Mentorship)
A diferencia de otras herramientas de automatización, AI-Haklab Pro está diseñado para **formar hackers**. El motor no solo sugiere comandos, sino que actúa como un **Dæmon Mentor** que explica:
*   **Anatomía del Exploit:** Por qué el payload funciona en esa arquitectura específica.
*   **Evasión de Defensas:** Técnicas para bypass de WAF, IDS y EDR.
*   **Ciclo OODA:** Metodología táctica de observación, orientación, decisión y acción.

---

## 🛠️ Características Principales
*   🧠 **Persistent Memory (Engram):** Retención de vectores de ataque y hallazgos entre sesiones de hacking.
*   👨‍🏫 **Modo Mentor:** Explicaciones detalladas en jerga hacker para elevar el nivel técnico del operador.
*   📋 **Skill Registry:** Módulos especializados en Recon, Exploit, Report y MCP.
*   🛡️ **Proactive Snapshots:** Sistema de respaldo automático de configuraciones críticas.
*   🎙️ **Voz Táctica:** Integración con TTS para alertas de estado del sistema en tiempo real.

---

## 🛠 Instalación y Requerimientos

### Requisitos Técnicos
1.  **Termux:** (Recomendado versión de F-Droid).
2.  **Entorno i-Haklab:** El agente está optimizado para las herramientas del laboratorio de **ivam3**.
3.  **Engram:** Para habilitar la memoria persistente.
4.  **Termux-API:** Para la síntesis de voz táctica.

### Despliegue Rápido
```bash
git clone https://github.com/kuromi04/AI-Haklab.git
cd AI-Haklab
chmod +x install.sh
./install.sh
```

---

## 📖 Guía de Operación
Para iniciar el motor de inteligencia:
```bash
AI-Haklab
```
### Comandos del Sistema
*   `/menu`: Gestión de proveedores de IA (Google, OpenAI, Anthropic).
*   `/update`: Sincronización con las últimas tácticas del repo.
*   `/backup`: Backup manual de la bóveda de configuración.
*   `/clean`: Purga de logs temporales e informes de intrusión.
