# 🕵️ AI-Haklab Pro v17.5: Tactical Hacker Engine
> **Pionero absoluto en la adaptación de ecosistemas de IA avanzada para el Pentesting Nativo en Termux.**

## 🏆 Un Hito en la Ciberseguridad Móvil
Este proyecto marca un precedente histórico: **@kuromi04** es el primer desarrollador en portar, adaptar y evolucionar los conceptos de arquitectura de IA (inspirados en [Gentle-AI](https://github.com/Gentleman-Programming/gentle-ai.git)) para que funcionen de forma 100% nativa y optimizada dentro del entorno **Termux**. 

AI-Haklab Pro no es solo un script; es la primera implementación móvil de un agente con memoria persistente, registro de habilidades y una filosofía de mentoría hacker.

## 🚀 Innovaciones Clave (Powered by Gentle-AI Philosophy)
He adaptado los pilares de Gentle-AI para la terminal móvil más potente:
*   🧠 **Termux Engram Sync:** Primer sistema de memoria persistente para hacking en dispositivos móviles.
*   📋 **Mobile Skill Registry:** Librería modular de tácticas de pentesting (`skills/`) optimizada para procesadores ARM.
*   👨‍🏫 **Hacker Mentor Persona:** El dæmon táctico que educa al operador en tiempo real, transformando la experiencia de aprendizaje en Termux.
*   🛡️ **Snapshots Proactivos:** Protección de datos y configuraciones en el almacenamiento local de Android.

---

## 🛠 Instalación y Requerimientos

### Requisitos Técnicos
1.  **Termux:** Recomendado versión siempre desde su github oficial: [termux-app](https://github.com/termux/termux-app).
2.  **i-Haklab Environment:** Optimización exclusiva para las herramientas del laboratorio de **Ivam3**.
3.  **Engram CLI:** Para la persistencia de inteligencia táctica.
4.  **Termux-API:** Para la síntesis de voz táctica.

### Despliegue de Inteligencia
```bash
git clone https://github.com/kuromi04/AI-Haklab.git
cd AI-Haklab
chmod +x install.sh
./install.sh
```

---

## 📖 Operación del Sistema
Ejecuta el motor:
```bash
AI-Haklab
```
*   `/menu`: Configuración de modelos (DeepSeek, Gemini, Anthropic).
*   `/update`: Sincronización de tácticas con el cuartel general.
*   `/backup`: Snapshot manual de la base de conocimientos.
